✂️ Simulador de Turnos - Peluquería "Tu Estilo"

🧾 Descripción general
Este proyecto es un simulador interactivo de turnos para una peluquería, desarrollado como parte de la Entrega 2 del curso de JavaScript. Permite que el usuario reserve un turno seleccionando nombre, servicio y horario. El sistema informa si un horario está disponible o no, y guarda todos los datos en el almacenamiento local del navegador.

⚙️ Funcionalidades
- Formulario para ingresar nombre, seleccionar servicio y horario.
- Campo de horario dividido en hora y minutos con salto automático.
- Validación de turnos duplicados: no permite reservar dos veces el mismo horario.
- Confirmación visual al reservar turno.
- Lista completa de horarios disponibles y no disponibles.
- Turnos guardados en localStorage para persistencia entre sesiones.
